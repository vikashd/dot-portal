DOT Drop is a secured data sharing application where you can safely store and easily share your data with encryption.

How does DOT Drop work

DOT Drop conceals data by encrypting files. Encryption is converting data into a code to prevent unauthorized access.

Users can decrypt data files with only token presented when folder was created.

Start with creating folder

You can create an airtight data folder by clicking New button.

If youâ€™ve already got a token shared by someone, Import button will help you add the folder to your list.

Keep your token safe

Once you create a folder, DOT Drop will grab you a token which is basically a passcode of the folder created. You can share this token with someone else who also wants to access the files.

Remember that you have to copy and save this token somewhere safe. There is no way to access the folder if you lose it.

Make a drop

You can add and also remove data once you unlock the folder with token.

A different permission level on files will be applied depending on who created the folder. Owner of the folder is able to rename the folder, regenerate the token and delete data files.

Time out

The unlocked folder will be locked again when you go back to your drop list.

But even if you forgot and left it open, DOT Drop will secure your data by locking the folder automatically after a certain period of inactivity.
