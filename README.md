# Dot portal

## Built using

-   [Jekyll](https://jekyllrb.com)
-   [Bootstrap](https://getbootstrap.com/)

## Setup

### Requirements

-   Ruby >= 2.5
-   Node >= 14

### Installation

```
$ bundle install
$ npm install && npm run serve
```

Open from http://localhost:4000/dot-portal

### Deployment/build

```
$ npm run build
```

Static files generated in `docs` folder
